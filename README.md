## About Me

我是一个以 **情报分析** 与 **安全实战** 为核心能力的技术从业者。  
擅长从复杂、分散、噪声较高的信息中提炼关键线索，并结合渗透测试与技术分析手段完成验证、归因与路径还原。

我的工作方法偏向：

- **evidence-driven**：以证据链为基础推进分析
- **offense-informed**：从攻击者视角理解系统与行为
- **trace-oriented**：重视时间线、关系网与行为轨迹
- **practical security**：关注真实环境中的可利用性与风险落点
